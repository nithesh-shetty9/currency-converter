# 💱 Currency Converter

A simple and responsive **Currency Converter** built using **HTML, CSS, and JavaScript**. It fetches real-time exchange rates using **CurrencyAPI** and allows users to convert between different currencies instantly.

🌐 **Live Demo:**  
https://nithesh-shetty9.github.io/currency-converter/

---

## ✨ Features

- 🌍 Convert between 160+ international currencies
- ⚡ Fetches real-time exchange rates using CurrencyAPI
- 💱 Convert between any supported currency pair
- 📈 Displays the current exchange rate
- 💰 Supports decimal input values
- 📱 Responsive and user-friendly interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- CurrencyAPI (REST API)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/nithesh-shetty9/currency-converter.git
```

### Navigate to the Project

```bash
cd currency-converter
```

### Run the Project

Open `index.html` in your preferred web browser.

For development, you can also use the **Live Server** extension in VS Code.

---

## 🔑 API Configuration

This project uses **CurrencyAPI** to fetch live exchange rates.

1. Create an account on CurrencyAPI.
2. Generate your API key.
3. Replace the API key in `script.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

---

## 📖 How It Works

1. Select the base currency.
2. Select the target currency.
3. Enter the amount to convert.
4. The application requests the latest exchange rate from CurrencyAPI.
5. The converted amount and the current exchange rate are displayed instantly.
